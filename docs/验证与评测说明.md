## 检索评测

`evaluation/datasets/project_baseline.jsonl` 仅包含公开项目知识和不可回答问题，不包含私人简历事实。先将项目文档导入并完成向量索引，再运行：

```bash
cd backend
python scripts/evaluate_retrieval.py \
  --dataset evaluation/datasets/project_baseline.jsonl \
  --user-id ADMIN_USER_ID \
  --knowledge-base-id KNOWLEDGE_BASE_ID
```

报告包含 Recall@K、可回答命中率和不可回答召回率。单元测试中的 100% 是 Fake Retriever 的框架自测，不是实际模型成绩，不能用于简历或上线结论。

## 并发基线

```bash
cd backend
python scripts/load_test.py --base-url http://localhost:8080 --requests 200 --concurrency 20
```

脚本输出吞吐、失败数和 P50/P95/P99。当前 Windows 后台终端无法稳定采集本地压测输出，因此本轮不记录虚假性能数字；应在部署服务器或 CI 独占 runner 上执行，并同时监控 CPU、内存、数据库连接和日志吞吐。
